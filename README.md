\# CareerMate AI



CareerMate AI is a Flask-based AI career guidance chatbot for students.



\## Features



\- AI career guidance chatbot

\- Login and register system

\- Dashboard

\- Resume PDF analyzer

\- Skill roadmap generator

\- Interview practice

\- Chat history

\- Admin panel

\- Professional sidebar UI



\## Tech Stack



\- Python

\- Flask

\- SQLite

\- HTML

\- CSS

\- Gemini API



\## Run Locally



```bash

pip install -r requirements.txt

python app.py
